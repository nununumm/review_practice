\# 第2問：ユーザー一覧と投稿件数取得（N+1問題）



\*\*📅 出題日\*\*: 2026年6月28日



\## ❌ NGポイント (bad/ 配下)

1\. \*\*N+1問題\*\*: `findAll()` で全件取得後、ループの中で `user.getPosts().size()` を呼ぶことで、データ量(N)に比例して大量のSQLが発行され、システムがダウンする原因になる。



\## ⭕️ 改善案 (good/ 配下)

1\. ループで都度DBにアクセスするのをやめる。

2\. `UserRepository` に `@Query` を記述し、`LEFT JOIN` と `COUNT` を使って、\*\*DB側で集計した結果を1発のSQLで\*\*持ってくる。

3\. 結果を受け取るための専用のクラス (`UserPostCountDto`) を作成する。
